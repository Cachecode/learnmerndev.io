MERN Stack Web Application Tutorial


MERN is a Javascript Stack that is used for quick deployment of full-stack web applications. .


Get Started Learn More

What makes a MERN stack?

MERN utilizes 4 very powerful technologies: MongoDB, Express, React and Node.js. Designed to make the development process smoother and easier.

MongoDB

MongoDB is a cross-platform NoSQL database program.

Express.js

Express is a back end framework for Node.js.

React.js

React is a front-end JavaScript library.

Node.js

Node.js is a back-end JavaScript runtime environment.

Tutorials for building and using the MERN stack

To begin learning more about the MERN stack you can follow these tutorials crafted specifically for beginners. The "CREATE STACK" button will lead you to a tutorial for creating a MERN stack on a VM instance in the Google Cloud Platform. The "CREATE WEB APP" button will lead you to a tutorial for using this MERN stack to make a simple "Hello World" web application. Click the buttons to begin the tutorial of your choosing.
